#Sistema de eventos bancarios con React + Node.js

Proyecto que simula un flujo de transacciones bancarias en tiempo real
usando WebSockets, React (Vite) para el frontend y Express.js para el backend.

##Tecnologías
- Frontend: React + TailwindCSS + Vite
- Backend: Node.js + Express + WS
- Comunicación: WebSockets
- Generación de IDs: UUID
